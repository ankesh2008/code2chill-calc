# code2chill-calc
Scientific Calculator (Streamlit)

This project is a modern scientific calculator built with Streamlit, designed to replicate the look and feel of the Windows scientific calculator while running entirely in the browser.

It combines a clean dark UI, intuitive button layout, and powerful mathematical functions into a lightweight web app.

✨ Features
🔢 Basic arithmetic operations (addition, subtraction, multiplication, division)
📐 Scientific functions:
Trigonometry (sin, cos, tan)
Logarithmic functions (log, ln)
Exponential and powers (x², xʸ, exp)
Square root, absolute value, factorial
🔄 DEG / RAD angle mode switching
🧠 Expression evaluation with error handling
🎯 Clean and responsive button layout
🎨 Dark theme inspired by the Windows calculator
⚡ Real-time display updates
🎨 UI Design

The interface is styled to mimic a native desktop calculator experience:

Minimal dark theme with flat design
Clearly grouped function, operator, and number buttons
Highlighted "=" button for primary action
Compact and user-friendly layout
🛠️ Built With
Streamlit – for building the interactive UI
Python (math module) – for scientific computations
Custom CSS – for styling and layout control
🚀 How to Run
pip install streamlit
streamlit run app.py
📌 Use Case

This project is ideal for:

Learning how to build interactive apps with Streamlit
Practicing UI/UX design with custom CSS
Creating lightweight tools for quick calculations
